# Static Lens Photography Portfolio Website & Booking System

Professional, cinematic photography for South Africa's youth market.

## Features
- **Responsive Portfolio Website:** Showcasing packages and work with a cinematic black & gold aesthetic.
- **Online Booking System:** Custom-built HTML form with package selection, date/time picker, and add-ons.
- **Booking Backend:** Express.js server with SQLite database for managing client bookings.

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript.
- **Backend:** Node.js, Express.js.
- **Database:** SQLite (via `better-sqlite3`).

## Setup & Running

### Prerequisites
- Node.js installed.

### Local Development
1. Clone the repository.
2. Install dependencies for the booking server:
   ```bash
   cd booking-server
   npm install
   ```
3. Start the booking server:
   ```bash
   node server.js
   ```
4. Open `index.html` in your browser.

## Project Structure
- `index.html`: Main portfolio website.
- `booking.html`: Online booking form.
- `style.css`: Site-wide styling.
- `script.js`: Site-wide interactivity.
- `assets/`: Image assets (logos, etc.).
- `booking-server/`: Backend server for the booking system.
